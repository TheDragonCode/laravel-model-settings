# Справочник API

## Трейт HasSettings

| Метод | Возвращает | Назначение |
|-------|------------|------------|
| `settings()` | `SettingsService` | Доступ к итоговым настройкам этой модели |
| `defaultSettings()` | `SettingsService` | Доступ к общим значениям по умолчанию для класса модели |
| `modelSettings()` | Eloquent `Relation` | Загрузка значений по умолчанию и переопределений как связи |

Используйте связь `modelSettings` с `with()`, `load()` или `loadMissing()`. Для чтения и изменения
значений используйте два метода сервиса. Во время выполнения связь представлена пакетным классом
`SettingsRelation`, основанным на связи Laravel `MorphMany`.

## SettingsService

| Метод | Возвращает | Поведение |
|-------|------------|-----------|
| `all()` | `Collection` | Возвращает значения по умолчанию, объединённые с переопределениями модели |
| `get(int\|string\|UnitEnum $key)` | `mixed` | Возвращает переопределение, значение по умолчанию или `null` |
| `set(int\|string\|UnitEnum $key, mixed $value)` | `void` | Создаёт, заменяет или удаляет пустую настройку |
| `forget(int\|string\|UnitEnum $key)` | `void` | Удаляет настройку, если она существует |

Методы с ключом принимают backed enum и pure unit enum. Laravel преобразует backed enum в его
базовое значение, а pure unit enum — в имя варианта.

## Матрица выбора значения

| Переопределение модели | Значение класса по умолчанию | Результат `get()` | Включено в `all()` |
|------------------------|------------------------------|-------------------|--------------------|
| Есть | Есть | Переопределение | Переопределение |
| Есть | Нет | Переопределение | Переопределение |
| Нет | Есть | Значение по умолчанию | Значение по умолчанию |
| Нет | Нет | `null` | Нет записи |

Для несохранённой модели `get()` возвращает `null`, а `all()` — пустую коллекцию. Значения класса по
умолчанию наследуются только сохранёнными моделями.

## all

```php
$settings = $user->settings()->all();

$timezone = $settings->get('timezone');
```

Результат — `Illuminate\Support\Collection` с ключами по ключам настроек. Для настроек модели
переопределения заменяют значения по умолчанию с тем же ключом.

## get

```php
$timezone = $user->settings()->get('timezone');
```

Результат содержит итоговое декодированное или преобразованное значение. Если переопределения нет,
используется значение по умолчанию. Если нет ни переопределения, ни значения по умолчанию,
возвращается `null`.

## set

```php
$user->settings()->set('timezone', 'Europe/Paris');
```

Метод выполняет update-or-create для типа модели, идентификатора модели и ключа. Значение, которое
Laravel считает пустым, удаляет строку. В обоих случаях загруженная связь `modelSettings` очищается,
чтобы следующее чтение не использовало устаревшие данные.

## forget

```php
$user->settings()->forget('timezone');
```

Метод безопасен, если ключ отсутствует. Удаление переопределения не удаляет общее значение по
умолчанию. После удаления загруженная связь очищается.

## defaultSettings

Сервис, возвращаемый `defaultSettings()`, предоставляет те же четыре метода:

```php
$defaults = (new User)->defaultSettings();

$defaults->set('timezone', 'UTC');
$timezone = $defaults->get('timezone');
$all = $defaults->all();
$defaults->forget('timezone');
```
