# Даведнік API

## Трэіт HasSettings

| Метад | Вяртае | Прызначэнне |
|-------|--------|-------------|
| `settings()` | `SettingsService` | Доступ да выніковых налад гэтай мадэлі |
| `defaultSettings()` | `SettingsService` | Доступ да агульных значэнняў па змаўчанні для класа мадэлі |
| `modelSettings()` | Eloquent `Relation` | Загрузка значэнняў па змаўчанні і перавызначэнняў як сувязі |

Выкарыстоўвайце сувязь `modelSettings` з `with()`, `load()` або `loadMissing()`. Для чытання і
змянення значэнняў выкарыстоўвайце два метады сэрвісу. Падчас выканання сувязь прадстаўлена класам
пакета `SettingsRelation`, заснаваным на сувязі Laravel `MorphMany`.

## SettingsService

| Метад | Вяртае | Паводзіны |
|-------|--------|-----------|
| `all()` | `Collection` | Вяртае значэнні па змаўчанні, аб’яднаныя з перавызначэннямі мадэлі |
| `get(int\|string\|UnitEnum $key)` | `mixed` | Вяртае перавызначэнне, значэнне па змаўчанні або `null` |
| `set(int\|string\|UnitEnum $key, mixed $value)` | `void` | Стварае, замяняе або выдаляе пустую наладу |
| `forget(int\|string\|UnitEnum $key)` | `void` | Выдаляе наладу, калі яна існуе |

Метады з ключом прымаюць backed enum і pure unit enum. Laravel пераўтварае backed enum у яго базавае
значэнне, а pure unit enum — у назву варыянта.

## Матрыца выбару значэння

| Перавызначэнне мадэлі | Значэнне класа па змаўчанні | Вынік `get()` | Уключана ў `all()` |
|-----------------------|------------------------------|---------------|---------------------|
| Ёсць | Ёсць | Перавызначэнне | Перавызначэнне |
| Ёсць | Няма | Перавызначэнне | Перавызначэнне |
| Няма | Ёсць | Значэнне па змаўчанні | Значэнне па змаўчанні |
| Няма | Няма | `null` | Няма запісу |

Для незахаванай мадэлі `get()` вяртае `null`, а `all()` — пустую калекцыю. Значэнні класа па
змаўчанні атрымліваюць толькі захаваныя мадэлі.

## all

```php
$settings = $user->settings()->all();

$timezone = $settings->get('timezone');
```

Вынік — `Illuminate\Support\Collection` з ключамі па ключах налад. Для налад мадэлі перавызначэнні
замяняюць значэнні па змаўчанні з тым жа ключом.

## get

```php
$timezone = $user->settings()->get('timezone');
```

Вынік змяшчае выніковае дэкадаванае або пераўтворанае значэнне. Калі перавызначэння няма,
выкарыстоўваецца значэнне па змаўчанні. Калі няма ні перавызначэння, ні значэння па змаўчанні,
вяртаецца `null`.

## set

```php
$user->settings()->set('timezone', 'Europe/Paris');
```

Метад выконвае update-or-create для тыпу мадэлі, ідэнтыфікатара мадэлі і ключа. Значэнне, якое
Laravel лічыць пустым, выдаляе радок. У абодвух выпадках загружаная сувязь `modelSettings`
ачышчаецца, каб наступнае чытанне не выкарыстоўвала састарэлыя даныя.

## forget

```php
$user->settings()->forget('timezone');
```

Метад бяспечны, калі ключ адсутнічае. Выдаленне перавызначэння не выдаляе агульнае значэнне па
змаўчанні. Пасля выдалення загружаная сувязь ачышчаецца.

## defaultSettings

Сэрвіс, які вяртае `defaultSettings()`, мае тыя ж чатыры метады:

```php
$defaults = (new User)->defaultSettings();

$defaults->set('timezone', 'UTC');
$timezone = $defaults->get('timezone');
$all = $defaults->all();
$defaults->forget('timezone');
```
